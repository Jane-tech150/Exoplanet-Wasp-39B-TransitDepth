# Exoplanet-Wasp-39B-TransitDepth
This project explores how exoplanets are detected using the transit method.

The program takes observational data from the MAST archive, converts the data from FITS format to CSV, and normalizes the stellar flux so it can be analyzed more easily. Using this processed data, the program estimates the transit depth, the small drop in a star’s brightness that occurs when a planet passes in front of it.

The goal of the project was to understand how astronomers analyze light curves and how a simple calculation can reveal the presence of a planet orbiting a distant star.
